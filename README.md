# OKTODO

OKTODO es una aplicación móvil enfocada en la organización de tareas, metas y eventos, integrando un sistema de motivación basado en gamificación y una mascota virtual llamada Okto que acompaña al usuario durante su progreso.

  <img src="media/okto.png" alt="Okto Mascot" width="300">
</p>

## Descripción

La aplicación busca resolver problemas comunes como la falta de organización, pérdida de motivación y dificultad para coordinar objetivos en grupo. OKTODO combina planificación personal, funciones sociales y elementos interactivos para convertir la productividad en una experiencia más dinámica.

## Funcionalidades

- Registro de tareas y eventos diarios o en fechas específicas.
- Visualización por prioridad y en formato calendario.
- Sistema de rachas y puntos para incentivar constancia.
- Modo de concentración (“Okto Work Time”) con apoyo visual y reducción de distracciones.
- Creación de grupos para planear metas y eventos compartidos.
- Perfil personalizable con opciones de apariencia para la mascota.
- Bloqueo de aplicaciones seleccionadas para reducir distracciones 

## Público Objetivo

Dirigida a usuarios que buscan mejorar su organización personal, mantener constancia en sus metas y colaborar con otras personas en objetivos compartidos.

## Stack Tecnológico

**Mobile**
- Kotlin / Java
- Android Studio
- Jetpack Components
- Material Design

**Backend y Servicios**
- Firebase Authentication
- Bloqueo de apps
- Mensajeria entre amigos
- Google authentication

**Arquitectura**
- MVVM
- Clean Architecture (en planeación)

## Escalabilidad

- Sistema avanzado de hábitos diarios.
- Sincronización en la nube.
- Ranking entre usuarios.
- Expansión de mascotas y elementos personalizables.
